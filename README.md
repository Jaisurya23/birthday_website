# 🎀 Birthday Gift Website for Gokul

## 📁 Folder Structure

```
birthday_project/
├── index.html        ← START HERE (Entry page with cat + yes/no game)
├── cake.html         ← Cake page with candles to blow
├── praise.html       ← "That's a good boy" page
├── letter.html       ← Love letter with typing animation
├── memories.html     ← Memory wall + video

├── gift.html         ← Surprise gift with suspense timer
├── final.html        ← Final page with confetti + typing name
│
├── gifs/             ← PUT YOUR GIFs HERE (already included if you copied them)
│   ├── hi.gif
│   ├── anger.gif
│   ├── play.gif
│   ├── tease.gif
│   ├── Happy.gif
│   ├── Happy Happy Happy Dancing GIF....gif
│   ├── Fun Pink Sticker by Minto Inc..gif
│   └── cat_gun.gif
│
└── photos/           ← PUT YOUR PHOTOS & VIDEO HERE
    ├── photo1.jpg    (or .png, .jpeg, .webp)
    ├── photo2.jpg
    ├── photo3.jpg
    ├── photo4.jpg
    └── video.mp4     (or .mov, .webm)
```

## 🚀 How to Set Up

1. **Copy your GIFs** into the `gifs/` folder  
   (The filenames must match exactly as listed above)

2. **Create a `photos/` folder** and put your photos inside:
   - Name them `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`
   - Add your video as `video.mp4` (or .mov or .webm)
   - They'll auto-load on the Memory Wall page!
   - You can also upload them manually on the page using the upload button

3. **Open `index.html`** in a browser — that's it! 🎉

## 🎭 Page Flow

```
index.html  →  cake.html  →  praise.html  →  letter.html
→  memories.html  →  gift.html  →  final.html
```

### Special Features:
- **Yes/No game**: Say No 3 times and watch the cat reactions escalate 😈
- **Cake**: Make a wish before blowing candles! Auto-transitions after popup
- **Letter**: Typing animation — words appear one by one like a real letter
- **Memory Wall**: Photos auto-load from `photos/` folder + video player
- **Gift**: 3-second suspense countdown with progress ring before opening
- **Final**: "Gokul!" types out letter by letter before confetti explosion

## 💡 Tips

- Works best in Chrome or Edge
- Open with a local server if photos don't load automatically  
  (VS Code → Live Server, or just use the upload buttons on the page)
- All click effects: **sparks burst** wherever you click! ✨
- Pink & white theme throughout with floating hearts 💕
